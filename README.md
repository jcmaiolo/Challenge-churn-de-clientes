# Churn de Clientes

Projeto desenvolvido como parte do desafio **Telecom X** da Alura, com foco no processo de **ETL (Extração, Transformação e Carga)** e **Análise Exploratória de Dados (EDA)** utilizando Python.

## Objetivo
Analisar os fatores associados à evasão de clientes (churn), a partir de dados disponibilizados via API, gerando insights que auxiliem na compreensão do problema e apoiem decisões estratégicas.

## Estrutura do Projeto
- `TelecomX_BR.ipynb`: notebook principal com as etapas de extração, tratamento, análise exploratória e relatório final.

## Como executar
1. Abra o arquivo `TelecomX_BR.ipynb` no Google Colab.
2. Execute as células do notebook **do início ao fim**, na ordem apresentada.

## Fonte dos dados
Os dados são obtidos a partir de uma **API pública**, no formato **JSON**, conforme disponibilizado no desafio Telecom X.
